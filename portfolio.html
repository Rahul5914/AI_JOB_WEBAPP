What It Does

Six AI-powered
modules working together

📄
Resume Parser
NLP-based extraction of skills, experience, education, and contact details from PDF/DOCX resumes. Converts extracted data into 384-dim sentence embeddings.
spaCy Sentence Transformers pdfplumber Regex
🌐
Job Scraper
Collects real-time job listings from LinkedIn, Indeed, and Naukri. Converts job descriptions into embeddings for semantic comparison with resumes.
Selenium BeautifulSoup Requests Playwright
🎯
Recommendation Engine
Ranks jobs using cosine similarity between resume and job embeddings. Combines embedding similarity (60%) with skill overlap score (40%) for precise matching.
Cosine Similarity NumPy Embeddings ML Ranking
📊
ATS Score Analyzer
Compares resume against job descriptions, identifies missing skills, and provides AI-generated suggestions to improve ATS score and beat applicant filters.
Keyword Matching Gap Analysis TF-IDF NLP
🤖
RAG Career Assistant
Retrieval-Augmented Generation system that answers career questions by fetching relevant knowledge from a vector database and generating personalized advice with an LLM.
RAG Vector DB LLM Chroma
⚡
Auto Apply Bot
Browser automation bot that fills application forms and submits them automatically. Includes a ML feedback loop that learns from clicks and applications to improve recommendations.
Selenium Playwright ML Feedback Automation

Quick Start

Deploy in 3 steps

bash — ai-job-finder
# Step 1: Clone the repository
~$ git clone https://github.com/YOUR_USERNAME/ai-job-finder.git
Cloning into 'ai-job-finder'... done.

# Step 2: Install dependencies
~/ai-job-finder$ pip install -r requirements.txt
Successfully installed streamlit-1.35.0 ✓

# Step 3: Run locally
~/ai-job-finder$ streamlit run app.py
✓ App running at http://localhost:8501

# Or deploy to Streamlit Cloud — it's free!
~/ai-job-finder$ git push origin main # then go to share.streamlit.io
✓ Deployed!
